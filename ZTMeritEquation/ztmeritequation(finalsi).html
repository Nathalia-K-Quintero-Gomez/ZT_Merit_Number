ZT Merit Equation for Thermoelectric Pipe using Nanofluid

\[ZT= \frac{\left(-V_{\text{th}}/\Delta T\right)^2 \sigma}{K_{\text{tp}}} T\]

About this work

The presented website aims to provide a new approach in the calculation of ZT figure merit number of thermoelectric pipes by using nanofluid. More information about this new empirical equation can be found in the following journal: Estimations on Properties of Redox Reactions to Electrical Energy and Storage Device...
